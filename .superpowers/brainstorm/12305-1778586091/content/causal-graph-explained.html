因果关系有向图：形态与构建

用 causal_relations.xlsx 中的真实数据来解释

什么是有向图？

有向图就是一组节点（指标）和带箭头的连线（影响关系）。箭头方向表示"谁影响谁"。

类比：就像一棵家族树，"父母 → 孩子"表示父母影响了孩子的存在。这里是"因子指标 → 结果指标"表示因子影响了结果。

怎么构建？从 Excel 到图

Excel 中的一行数据：
结果指标名称 类别 因子指标名称
门诊人次 分类二 普通门诊就诊人次
变成图中的一条边：
普通门诊就诊人次 ——(分类二)——→ 门诊人次
构建过程（伪代码）：
# 遍历 Excel 每一行
for row in excel_data:
  # 添加一条边：因子 → 结果
  graph.add_edge(
    from: row["因子指标名称"],
    to: row["结果指标名称"],
    label: row["类别"]
  )

# Excel有49行 → 图中有49条边
# 连接了约30个不同的指标节点
